# Advent of Code 2025

Solutions and tools for [Advent of Code 2025](https://adventofcode.com/2025), including a CLI, helper class, and automated testing.

## Usage

To use automated data retrieval and solution submission, populate .env with your session cookie from Advent of Code.

To start a new day:
```bash
uv run task create {day}
```

To run the solution for a day:
```bash
uv run task day {day}
Part 1: 1234
Part 2: 5678
```
